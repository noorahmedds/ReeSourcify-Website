# REESourcify Static Website

A responsive static landing page built from the REESourcify pitch deck.

## Files
- `index.html` — website markup
- `styles.css` — responsive styling and animations
- `script.js` — mobile navigation behavior

## Run locally
Open `index.html` in a browser, or serve the folder with:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Deploy
Upload the folder to Netlify, Vercel, GitHub Pages, or any static hosting provider.
